# Emmet 量化交易實戰：從零理解並使用交易系統

這是一本文字與實作並重的新手教材，配套專案為相鄰目錄的
`../emmet-qt-bt1`。本書不另外發明一套簡化交易引擎；所有概念最終都要回到
配套系統的真實模型、資料、測試與操作入口。

## 寫作目標

- 讓沒有量化交易經驗的讀者理解市場、資料、訂單、撮合、會計與風控。
- 讓會寫一點 Python 的讀者能安全地安裝、檢查並操作配套系統。
- 解釋系統為何採用 Decimal、注入時鐘、事件全序、雙錢包與策略沙箱等設計。
- 由歷史資料與回測逐步前進，不把回測績效誤寫成未來獲利承諾。
- 明確標示已實作功能、設計中功能與尚未支援功能。

## 目前基線

本書依 `emmet-qt-bt1` 的權威設計文件撰寫。盤點時配套專案版本為
`0.3.0`，Phase 1–3 已完成，Phase 4 正在開發；資料管線、基礎模型、撮合、
執行與會計已有實作，完整引擎、Backtest Server、MCP、Paper 與 Live 則須依
各 Phase 實際完成狀態更新書稿。

> 本書是工程與研究教材，不構成投資建議。實盤交易可能損失全部本金；在讀者
> 完成資料驗證、回測、模擬盤、風控與操作演練前，不引導其投入真實資金。
